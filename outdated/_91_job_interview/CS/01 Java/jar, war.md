### JAR(Java Archive) 

- 클래스파일과 같은 자바 리소스 파일 + 라이브러리
- 자바 프로젝트를 압축한 파일로 이해
- JRE만 가지고도 실행 가능

### WAR(Web Application Archive)

- 웹 애플리케이션 압축 포맷
- servlet 등 웹 관련 자원만 포함
- WAR파일을 실행하려면 Tomcat 등의 웹 서버 (WEB)또는 웹 컨테이너(WAS)가 필요

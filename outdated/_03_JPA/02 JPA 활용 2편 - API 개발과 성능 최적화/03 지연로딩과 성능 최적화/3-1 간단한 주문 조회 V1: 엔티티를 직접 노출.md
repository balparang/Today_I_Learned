- Lazy Loading 시 먼저 프록시 객체를 만들어두고, 실제 객체를 사용할 때 Loading
- Jackson 이 순수한 Member를 뽑으려고 시도했으나, 실제로는 프록시 객체를 마주쳤기 때문에 `Type Definition Error`가 발생한 것이다.

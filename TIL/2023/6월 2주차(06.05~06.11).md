### REST API

- `REST API`
  - REST 아키텍처를 따르는 API
  - 필요한 자원과 행위를 HTTP 요청에 나타냄으로써, 요청 형식만 보고도 어떤 작업인지 추론 가능 👉여러 개발자들과 협업을 가능케함  
- `REST` 
  - `Representational State Transfer` 의 약자
  - API 작동방식에 조건을 부과하는 소프트웨어 아키텍처
  - `균일한 인터페이스`, `무상태`, `캐시 가능성`등의 원칙을 가짐
